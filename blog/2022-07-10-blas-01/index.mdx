## BLAS について
BLAS は線形代数の演算を行うライブラリ。

BLAS は 3 つのレベルに分類される

|レベル |概要                |
|:-----:|:------------------:|
|Level 1|ベクトル同士の計算  |
|Level 2|ベクトルと行列の計算|
|Level 3|行列と行列の計算    |

関数の先頭は接頭辞から始まり、以下のような意味がある。

|接頭辞|意味|型  |
|:----:|:--:|:--:|
|s     |単精度浮動小数点数|float|
|d    |倍精度浮動小数点数|double|
|c|単精度複素数||
|z|倍精度複素数||

:::note

以降の例で使用するため単精度浮動小数点数のベクトルを表示する関数として `disp_svec` を作っておきました。

```mdx-code-block
<Tabs>
<TabItem value="コード">
```

```c title="disp.c"
#include <stdio.h>
#include <string.h>

#define MAX_STR 10000
#define LEN_NUM 50

void disp_svec(float *X, int n){
    char str[MAX_STR] = {};
    char tmp[LEN_NUM] = {};
    int i;
    for(i = 0; i < n; i++){
        sprintf(tmp, "%8.2f ", X[i]);
        strcat(str, tmp);
    }
    puts(str);
}
```

## Level 1

### 定数倍して足す (cblas_*axpy)

$$\begin{align}
\boldsymbol{y} &\leftarrow \alpha\boldsymbol{x} + \boldsymbol{y}
\end{align}$$

#### 定義
```c
void cblas_saxpy(const int N, const float alpha, const float *X,
                 const int incX, float *Y, const int incY);
void cblas_daxpy(const int N, const double alpha, const double *X,
                 const int incX, double *Y, const int incY);
void cblas_caxpy(const int N, const void *alpha, const void *X,
                 const int incX, void *Y, const int incY);
void cblas_zaxpy(const int N, const void *alpha, const void *X,
                 const int incX, void *Y, const int incY);
```

|仮引数|意味|
|:----:|:--:|
|N|ベクトルの要素数|
|alpha|係数 $\alpha$|
|X|ベクトル $\boldsymbol x$|
|incX|$\boldsymbol x$ のインクリメント幅|
|Y|ベクトル $\boldsymbol y$|
|incY|$\boldsymbol y$ のインクリメント幅|

```mdx-code-block
<Tabs>
<TabItem value="コード">
```

```c title="ex01.c"
#include <cblas.h>
#include <stdio.h>
#include "disp.h"

int main(void){
    float X[] = {1, 2, 3, 4, 5};
    float Y[5] = {};

    cblas_saxpy(5, 3, X, 1, Y, 1);

    disp_svec(Y, 5);

    return 0;
}
```

## Level 2
### 行列とベクトルの積 (cblas_*gemv)

$$\begin{align}
\boldsymbol y &\leftarrow \alpha A \boldsymbol x + \beta \boldsymbol y\\
\boldsymbol y &\leftarrow \alpha A^T\boldsymbol x + \beta \boldsymbol y
\end{align}$$

#### 定義
```c
void cblas_sgemv(const CBLAS_LAYOUT layout,
                 const CBLAS_TRANSPOSE TransA, const int M, const int N,
                 const float alpha, const float *A, const int lda,
                 const float *X, const int incX, const float beta,
                 float *Y, const int incY);
void cblas_dgemv(CBLAS_LAYOUT layout,
                 CBLAS_TRANSPOSE TransA, const int M, const int N,
                 const double alpha, const double *A, const int lda,
                 const double *X, const int incX, const double beta,
                 double *Y, const int incY);
void cblas_cgemv(CBLAS_LAYOUT layout,
                 CBLAS_TRANSPOSE TransA, const int M, const int N,
                 const void *alpha, const void *A, const int lda,
                 const void *X, const int incX, const void *beta,
                 void *Y, const int incY);
void cblas_zgemv(CBLAS_LAYOUT layout,
                 CBLAS_TRANSPOSE TransA, const int M, const int N,
                 const void *alpha, const void *A, const int lda,
                 const void *X, const int incX, const void *beta,
                 void *Y, const int incY);
```

|仮引数|意味|
|:----:|:--:|
|layout|レイアウト (`CblasRowMajor`, `CblasColMajor`)|
|TransA|転置するかどうか (`CblasNoTrans`, `CblasTrans`, `CblasConjTrans`)|
|M|$A$ の行数|
|N|$A$ の列数|
|alpha|定数 $\alpha$|
|A|行列 $A$|
|lda|$A$ の列数 (行数)|
|X|ベクトル $\boldsymbol x$|
|incX|$\boldsymbol x$ のインクリメント幅|
|beta|定数 $\beta$|
|Y|ベクトル $\boldsymbol y$|
|incY|$\boldsymbol y$ のインクリメント幅|

```mdx-code-block
<Tabs>
<TabItem value="コード 1">
```

```c title="ex08.c"
#include <cblas.h>
#include <stdio.h>
#include "disp.h"

int main(void){
    float A[] = {
        2, 3, 1,
        1, 2, 3
    };

    float X[] = {3, 2, 1};
    float Y[2] = {};

    cblas_sgemv(CblasRowMajor, CblasNoTrans, 2, 3, 1, A, 3, X, 1, 1, Y, 1);

    disp_svec(Y, 2);

    return 0;
}
```
