C 言語で BLAS を使う

2022年7月10日 · 約12分

BLAS について​

BLAS は線形代数の演算を行うライブラリです。

BLAS は 3 つのレベルに分類されます。

レベル概要
Level 1ベクトル同士の計算
Level 2ベクトルと行列の計算
Level 3行列と行列の計算

関数の先頭は接頭辞から始まり、以下のような意味があります。

接頭辞意味型
s単精度浮動小数点数float
d倍精度浮動小数点数double
c単精度複素数
z倍精度複素数
note

以降の例で使用するため単精度浮動小数点数のベクトルまたは行列を表示する関数として disp_svec、disp_smat を作っておきました。

  • コード
  • ヘッダーファイル
disp.c
#include <stdio.h>
#include <string.h>

#define MAX_STR 10000
#define LEN_NUM 50

void disp_svec(float *X, int n){
    char str[MAX_STR] = {};
    char tmp[LEN_NUM] = {};
    int i;
    for(i = 0; i < n; i++){
        sprintf(tmp, "%8.2f ", X[i]);
        strcat(str, tmp);
    }
    puts(str);
}

void disp_smat(float *X, int M, int N){
    char str[MAX_STR] = {};
    char tmp[LEN_NUM] = {};
    int i, j;

    for(i = 0; i < M; i++){
        for(j = 0; j < N; j++){
            sprintf(tmp, "%8.2f ", X[i * N + j]);
            strcat(str, tmp);
        }
        strcat(str, "\n");
    }
    printf("%s", str);
}
disp.h
void disp_svec(float *X, int n);
void disp_smat(float *X, int M, int N);

Level 1​

定数倍して足す (cblas_*axpy)​

y←αx+y\begin{align} \boldsymbol{y} &\leftarrow \alpha\boldsymbol{x} + \boldsymbol{y} \end{align}y​←αx+y​​

定義​

void cblas_saxpy(const int N, const float alpha, const float *X,
                 const int incX, float *Y, const int incY);
void cblas_daxpy(const int N, const double alpha, const double *X,
                 const int incX, double *Y, const int incY);
void cblas_caxpy(const int N, const void *alpha, const void *X,
                 const int incX, void *Y, const int incY);
void cblas_zaxpy(const int N, const void *alpha, const void *X,
                 const int incX, void *Y, const int incY);
仮引数意味
Nベクトルの要素数
alpha係数 α\alphaα
Xベクトル x\boldsymbol xx
incXx\boldsymbol xx のインクリメント幅
Yベクトル y\boldsymbol yy
incYy\boldsymbol yy のインクリメント幅

  • Makefile
  • 実行例
ex01.c
#include <cblas.h>
#include <stdio.h>
#include "disp.h"

int main(void){
    float X[] = {1, 2, 3, 4, 5};
    float Y[5] = {};

    cblas_saxpy(5, 3, X, 1, Y, 1);

    disp_svec(Y, 5);

    return 0;
}
